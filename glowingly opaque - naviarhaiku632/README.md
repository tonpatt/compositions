# glowingly opaque - naviarhaiku632

**glowingly opaque - naviarhaiku632** The sound structure is defined in Csound, while the score was externally generated using Python.

## Audio Generation

This project uses a `Makefile` to automate the rendering process. To successfully build the audio files, ensure the following components are in the same directory:

* **Main files**: `drone1.csd`
* **Python**: `drone1.py` and `score.py`

### Commands

* **To create the FLAC file:**

```bash
make flac

```

* **To create the OGG file:**

```bash
make ogg

```

## Requirements

* **Csound 6.18** or higher
* **GNU Make**
* **Python 3.12** or higher
